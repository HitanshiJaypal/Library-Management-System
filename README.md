# 📚 Library Management System (Frontend Project)

A modern, responsive **Library Management System** built using **HTML, CSS, Bootstrap, and JavaScript**.  
This project allows users to manage books, issue/return them, and track history — all with a clean UI.

---

## 🚀 Features

- 🔐 Admin Login System (LocalStorage-based)
- 📊 Dashboard with live statistics
- 📖 View all books
- ➕ Add new books
- 📤 Issue books to users
- ↩️ Return issued books
- 📋 Issue history tracking
- 🔍 Search, filter, and sort books
- 🎨 Modern UI with animations and responsive design

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3 (Custom + Bootstrap 5)**
- **JavaScript (Vanilla JS)**
- **LocalStorage (for data persistence)**

## 💡 How It Works

- All data is stored in **browser LocalStorage**
- No backend required
- Works completely offline after first load

---

## ▶️ How to Run

1. Download or clone this repository
2. Open `login.html` in your browser
3. Login using default credentials
4. Start managing the library
